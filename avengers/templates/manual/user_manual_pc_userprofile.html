User Profile

　ここでは、User Profileタブでできることについて解説させていただきます。このタブでできることは、あなたのプロフィール情報を確認したり、変更したりすることです。ここ以外にもプロフィールの確認・変更画面はいくつかのページに設置してありますが、スマホをはじめ画面の横幅が600pxを下回りますとこのタブ以外ではプロフィールが表示されなくなります。ですので、このプロフィールタブは基本的にスマホユーザー向けになります。まずは以下の画像をご覧いただきましょう。

《表示内容》

　User Profileのホーム画面で表示される内容はプロフィール画像とユーザー名のみです。プロフィールの確認と変更を目的にしたページですのでそれ以外に表示される情報はありません。

《ボタン説明》

　User Profileのホーム画面に設置してあるボタンは4つです。

・プロフィール編集：　登録されているプロフィール内容を変更します。

・運営からのお知らせ：　運営からのお知らせを確認できます。基本的に全ユーザー対象のお知らせはこちらに表示されます。個々のユーザーを対象にする場合はダイレクトメッセージを送らせていただきます。例としては、あなたが投稿した記事が他のユーザーに通報され、運営側でも不適切と判断して削除した場合などです。

・利用規約確認：　利用規約を確認できます。規約の変更があった場合なんかにも告知いたしますのでご確認いただければと思います。

・プライバシーポリシー確認：　プライバシーポリシーを確認できます。こちらも中身を変更したときには告知いたします。
