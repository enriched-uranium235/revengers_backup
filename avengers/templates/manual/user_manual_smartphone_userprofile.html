User Profile

　ここでは、User Profileタブでできることについて解説させていただきます。このタブでできることは、あなたのプロフィール情報を確認したり、変更したりすることです。ここ以外にもプロフィールの確認・変更画面はいくつかのページに設置してありますが、スマホをはじめ画面の横幅が600pxを下回るデバイスをご利用していますとこのタブ以外ではプロフィールが表示されなくなります。ですので、このプロフィールタブは基本的にスマホユーザー向けになります。まずは以下の画像をご覧いただきましょう。

　上の画像はアカウント登録後一切プロフィールを変更しない状態でこのタブを開いた時の画面です。

《表示内容》

　User Profileのホーム画面で表示される内容はプロフィール画像とユーザー名のみです。プロフィールの確認と変更を目的にしたページですのでそれ以外に表示される情報はありません。

《ボタン説明》

　User Profileのホーム画面に設置してあるボタンは以下の一つだけです。

・プロフィール編集：　登録されているプロフィール内容を変更します。
